git init : Inicializar un nuevo proyecto proyecto.

git status : para ver el estado de nuestras archivos

git add : para agragar un archivo al staging area.

git commit -m "" : para crear un primer punto de control de nuestro código.

git log : para ver todos los commits que hemos creados

git checkout -- nombre de archivo : para revertir los de los archivos

git diff : para ver las diferencias hachas en los archivos

git push -u origin --nombre de la rama--

git branch 


.gitignore : archivo en donde se colocaran todos las archivos que no se deseen subir.
